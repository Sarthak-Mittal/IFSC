About

Get to know about this project and how it works.

This project help you get your bank details only using your IFSC code. By using this service, you can get the details of your

  • Bank name.
  • Bank contact number.
  • Bank branch detail location and address completed with Google Map link.
  • Availability of Payment and settlement systems like RTGS, NEFT, IMPS, and UPI.

We hope this project will help serve the purpose for people who might want to look up some information about their bank branch.

What is IFSC?

IFSC or Indian Financial System Code is an alpha-numeric code that uniquely identifies a bank-branch participating in the NEFT system. Itâ€™s a 11-digit code with the first 4 alpha characters representing the bank, and the last 6 characters representing the branch. The 5th character is 0 (zero).

IFSC is used by the NEFT system to identify the originating / destination banks / branches and also to route the messages appropriately to the concerned banks / branches.
