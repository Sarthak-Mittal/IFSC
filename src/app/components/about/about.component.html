About this project

This project help you get your bank details only using your IFSC code. By using this service, you can get the details of your

  • Bank name.
  • Bank contact number.
  • Bank branch detail location and address completed with Google Map link.
  • Payment and settlement systems like RTGS, NEFT, IMPS, and UPI.

We hope this project will help serve the purpose for people who might want to look up some information about their bank branch.

If you want to contribute to this project, please follow this link to see the repo.

What is IFSC?

IFSC is the Indian Financial System Code which consist of alphanumeric code that facilitates electronic funds transfer in India. A code uniquely identifies each bank branch participating in the three main Payment and settlement systems in India: the Real Time Gross Settlement (RTGS), National Electronic Fund Transfer (NEFT) and Immediate Payment Service (IMPS) systems. This project is using RazorPay API to fetch the bank data.

Razorpay is the only payments solution in India that allows businesses to accept, process and disburse payments with its product suite. It gives you access to all payment modes including credit card, debit card, netbanking, UPI and popular wallets including JioMoney, Mobikwik, Airtel Money, FreeCharge, Ola Money and PayZapp.
